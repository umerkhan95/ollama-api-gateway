# Ollama API Gateway - Logo Assets

This directory contains the logo and branding assets for the Ollama API Gateway.

## Available Assets

### 1. **logo.svg** (200x200px)
- Main logo with full details
- Use for: Large displays, hero sections, marketing materials
- Colors: Primary gradient (#4F46E5 to #7C3AED)

### 2. **logo-icon.svg** (40x40px)
- Simplified icon version
- Use for: App icons, small spaces, navigation bars
- Transparent background

### 3. **logo-horizontal.svg** (300x80px)
- Horizontal logo with text
- Use for: Headers, email signatures, light backgrounds
- Includes "Ollama API Gateway" text

### 4. **logo-horizontal-dark.svg** (300x80px)
- Dark theme version of horizontal logo
- Use for: Dark mode interfaces, dark backgrounds
- Lighter gradient (#60A5FA to #3B82F6)

### 5. **favicon.svg** (32x32px)
- Favicon for browser tabs
- Use for: Browser tab icons, bookmarks
- Square format with rounded corners

## Design Guidelines

- **Minimum Size**: Don't scale logos below 24x24px for the icon version
- **Clear Space**: Maintain at least 8px clear space around all logos
- **Background**: Logos work best on white or dark backgrounds
- **Modifications**: Please don't modify colors, proportions, or add effects

## File Formats

All logos are provided in SVG format for:
- ✅ Scalability without quality loss
- ✅ Small file sizes
- ✅ Easy color customization
- ✅ Crisp rendering on any display

## Need Different Formats?

If you need PNG or other formats, you can:
1. Open SVG files in any vector editing tool (Figma, Illustrator, Inkscape)
2. Export to your desired format and size
3. Use online converters like CloudConvert or SVG to PNG converters
